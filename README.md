# Smart Farm Monitoring System

스마트팜 모니터링 시스템 시뮬레이션

![타이틀](https://github.com/IndiaInk10/SmartFarmMonitoringSystem/assets/49700162/e7df0f06-d556-45df-9855-93dcc77ad57c)

## 요구사항

- 리눅스 기반 시스템
- C 또는 Python 기반 프로그램
- 모의 센서 데이터(무작위 또는 패턴)
- IPC 메커니즘
- Relational Database 사용
- MQTT를 통한 데이터 전송
